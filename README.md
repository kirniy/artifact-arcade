# ARTIFACT - AI-Powered Arcade Fortune Machine

An AI-powered arcade fortune-telling machine built on Raspberry Pi 4 with a multi-display system, camera, thermal printer, and Gemini AI integration.

## Quick Start

```bash
# Run simulator
PYTHONPATH=src:$PYTHONPATH python -m artifact.simulator.main
```

## Features

- 5 game modes: Fortune, Zodiac, Roulette, Quiz, AI Prophet
- Russian UI with Cyrillic bitmap fonts
- Animated mystical eye in crystal ball
- Thermal receipt printing with caricatures
- Gemini AI integration for predictions
