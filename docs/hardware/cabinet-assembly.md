# Инструкция по Сборке Корпуса ARTIFACT

## Необходимые Инструменты

### Основные
- Шуруповёрт / дрель
- Набор бит и свёрл
- Угольник столярный
- Рулетка 3м
- Струбцины (минимум 4 шт.)
- Киянка резиновая

### Расходные Материалы
- Саморезы 4×40мм — 100 шт.
- Саморезы 3×20мм — 50 шт.
- Уголки мебельные 40×40мм — 12 шт.
- Петли рояльные 400мм — 2 шт.
- Магнитные защёлки — 4 шт.
- Клей столярный ПВА — 1 банка
- Кромка ПВХ 18мм — 10 м

---

## Порядок Сборки

### Этап 1: Подготовка Деталей

**Время: ~1 час**

1. Проверить комплектность всех деталей по ведомости
2. Проверить размеры критических сопряжений
3. Разметить места крепления уголков и полок
4. Предварительно просверлить отверстия под саморезы (⌀3мм)

```
Контрольные точки:
□ Боковые панели симметричны
□ Передняя панель подходит по ширине (540мм)
□ Полки входят между боковинами (504мм)
```

---

### Этап 2: Сборка Основания

**Время: ~30 минут**

```
    ┌────────────────────────┐
    │      ОСНОВАНИЕ         │
    │      500×540мм         │
    └────────────────────────┘
           ↑         ↑
    ┌──────┘         └──────┐
    │                        │
    │  БОКОВАЯ    БОКОВАЯ   │
    │  ПАНЕЛЬ 1   ПАНЕЛЬ 2  │
    │                        │
```

**Шаги:**

1. Установить боковую панель 1 вертикально
2. Закрепить струбцинами к рабочему столу
3. Приложить основание к нижнему торцу боковины
4. Проверить угол 90° угольником
5. Закрепить уголками изнутри (2 шт.)
6. Повторить для боковой панели 2

**Крепёж:**
- 4 уголка 40×40мм
- 8 саморезов 4×40мм

---

### Этап 3: Установка Полки Основания

**Время: ~20 минут**

```
Высота установки: 150мм от пола
Размер полки: 464×440мм
```

**Шаги:**

1. Отмерить 150мм от основания на обеих боковинах
2. Разметить линию установки полки
3. Закрепить уголки на боковинах
4. Установить полку на уголки
5. Зафиксировать саморезами

**Крепёж:**
- 4 уголка 40×40мм
- 8 саморезов 3×20мм

---

### Этап 4: Установка Передней Панели (Нижняя часть)

**Время: ~30 минут**

```
Передняя панель устанавливается в несколько этапов:
1. Сначала нижняя часть (до панели управления)
2. Затем секция экрана (после установки панели управления)
```

**Шаги:**

1. Примерить переднюю панель к каркасу
2. Проверить вырезы под динамики и принтер
3. Нанести клей на торцы боковин (нижняя зона)
4. Прижать панель и зафиксировать струбцинами
5. Закрепить саморезами с шагом 150мм

**Крепёж:**
- Саморезы 4×40мм — 12 шт.
- Клей ПВА

---

### Этап 5: Установка Панели Управления

**Время: ~40 минут**

```
Угол установки: 12° от горизонтали
Высота переднего края: 950мм от пола
```

**Шаги:**

1. Изготовить клиновидные опоры для угла 12°
   - Или использовать скошенные бруски
2. Закрепить опоры на боковинах
3. Установить панель управления на опоры
4. Проверить угол уровнем/угломером
5. Закрепить саморезами через опоры

**Альтернатива:**
- Вырезать пазы в боковинах под углом 12°
- Вставить панель в пазы

**Крепёж:**
- Бруски-опоры: 2 шт.
- Саморезы 4×40мм — 6 шт.

---

### Этап 6: Установка Средней Полки

**Время: ~15 минут**

```
Высота установки: 500мм от основания
Размер полки: 464×200мм
Назначение: NovaStar T50
```

**Шаги:**

1. Отмерить высоту 500мм на боковинах
2. Закрепить уголки
3. Установить полку
4. Зафиксировать

**Крепёж:**
- 2 уголка 40×40мм
- 4 самореза 3×20мм

---

### Этап 7: Монтаж LED-Панелей

**Время: ~1 час**

```
LED-панели: 4× P3 64×64 (384×384мм итого)
Позиция: За окном в передней панели
```

**Шаги:**

1. Изготовить монтажную раму из брусков 20×20мм
2. Закрепить раму за передней панелью
3. Установить DH418 на раму
4. Разместить 4 LED-панели в сетке 2×2
5. Подключить HUB75E кабели
6. Проверить работоспособность до закрытия

```
Схема подключения:
┌─────┬─────┐
│ P2  │ P3  │  ← HUB2, HUB3
├─────┼─────┤
│ P1  │ P4  │  ← HUB1, HUB4
└─────┴─────┘
```

---

### Этап 8: Установка Секции Экрана (Верхняя передняя часть)

**Время: ~30 минут**

```
После монтажа LED-панелей
Угол наклона верхней части: 20° назад
```

**Шаги:**

1. Закрепить верхнюю часть передней панели
2. Обеспечить наклон 20° с помощью:
   - Скошенных боковин, или
   - Угловых вставок
3. Проверить, что окно экрана совпадает с LED-панелями
4. Зафиксировать

---

### Этап 9: Установка Секции Маркиза

**Время: ~20 минут**

```
Размер: 540×350мм
Содержит: Отверстие для камеры
```

**Шаги:**

1. Установить маркиз на верх конструкции
2. Обеспечить угол для стока воды (если для улицы)
3. Закрепить саморезами
4. Установить камеру в отверстие

---

### Этап 10: Установка Бегущей Строки

**Время: ~30 минут**

```
WS2812B: 32×8 (центр) + 8×8 (лево) + 8×8 (право)
Общий размер: 480×80мм
```

**Шаги:**

1. Изготовить крепёжную планку для LED-матриц
2. Закрепить планку за окном бегущей строки
3. Приклеить LED-матрицы к планке (двухсторонний скотч)
4. Подключить питание и данные (GPIO 21)

---

### Этап 11: Установка Задней Панели

**Время: ~30 минут**

```
Задняя панель — съёмная для обслуживания
Варианты крепления: петли или магниты
```

**Вариант А: Петли**

1. Закрепить рояльные петли на одной боковине
2. Навесить заднюю панель
3. Установить защёлки на противоположной стороне

**Вариант Б: Магниты**

1. Установить магнитные защёлки по периметру
2. Закрепить ответные пластины на задней панели
3. Добавить ручку или углубление для открывания

---

### Этап 12: Монтаж Электроники

**Время: ~2 часа**

**Порядок установки:**

1. **Блок питания** (основание)
   - Закрепить на полке основания
   - Подключить сетевой кабель
   - Вывести линии 5В

2. **Raspberry Pi 4** (над БП)
   - Закрепить на стойках или к полке
   - Подключить питание USB-C
   - Подключить HDMI к T50

3. **NovaStar T50** (средняя полка)
   - Закрепить на средней полке
   - HDMI от Pi
   - Ethernet к DH418

4. **Прокладка кабелей**
   - GPIO шлейф к панели управления
   - USB к большой кнопке
   - CSI шлейф к камере
   - Аудио к динамикам

---

### Этап 13: Финишная Отделка

**Время: ~3-4 часа**

1. **Шпаклёвка**
   - Заполнить щели и отверстия от саморезов
   - Дать высохнуть (2-4 часа)

2. **Шлифовка**
   - Обработать все поверхности P120-P180
   - Скруглить острые углы

3. **Грунтовка**
   - Нанести грунт в 1-2 слоя
   - Просушка между слоями

4. **Покраска**
   - Краска для дерева (акриловая или алкидная)
   - 2-3 слоя с промежуточной сушкой
   - Рекомендуемый цвет: чёрный матовый

5. **Кромка**
   - Наклеить ПВХ-кромку на видимые торцы
   - Обрезать излишки

---

## Проверочный Список

### После Этапа 4 (Каркас)
- [ ] Конструкция устойчива
- [ ] Все углы 90° (кроме специальных)
- [ ] Нет перекосов и люфтов

### После Этапа 8 (Передняя часть)
- [ ] Все окна на своих местах
- [ ] Вырезы соответствуют компонентам
- [ ] Панель управления под правильным углом

### После Этапа 11 (Корпус готов)
- [ ] Задняя панель открывается/закрывается
- [ ] Вентиляция достаточна
- [ ] Все полки на месте

### После Этапа 12 (Электроника)
- [ ] Все компоненты работают
- [ ] Кабели уложены и закреплены
- [ ] Нет замыканий и перегревов

### Финальная Проверка
- [ ] Автомат включается и работает
- [ ] Все режимы функционируют
- [ ] Звук воспроизводится
- [ ] Принтер печатает
- [ ] Камера снимает

---

## Советы и Рекомендации

1. **Всегда** проверяйте размеры компонентов перед вырезами
2. **Используйте** клей + саморезы для надёжности
3. **Не затягивайте** саморезы до упора сразу — сначала наживите все
4. **Тестируйте** электронику на каждом этапе, не дожидаясь конца сборки
5. **Маркируйте** провода и разъёмы для упрощения обслуживания
6. **Оставьте** запас кабелей для возможного снятия компонентов
7. **Документируйте** любые отклонения от чертежей
