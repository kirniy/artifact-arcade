## Детальное Описание Деталей

### 1. Боковые Панели (×2)

```
Размер заготовки: 1750 × 500 мм
Форма: Сложный профиль (см. side-view.svg)

Особенности:
- Срез под 20° в верхней части (для наклона экрана)
- Скос для панели управления (12°)
- Сужение от 500мм (низ) до ~350мм (верх)

Обработка:
1. Вырезать прямоугольник 1750×500
2. Разметить профиль по шаблону
3. Вырезать лобзиком по контуру
4. Обработать кромки
```

### 2. Передняя Панель

```
Размер: 1200 × 540 мм

Вырезы (координаты от левого нижнего угла):
┌─────────────────────────────────────────┐
│  Бегущая строка: 490×90мм              │ Y=1050
│  Центр по ширине                        │
├─────────────────────────────────────────┤
│  Основной экран: 395×395мм             │ Y=600
│  Центр по ширине                        │
├─────────────────────────────────────────┤
│  Динамик L: ⌀70мм, X=165мм, Y=400мм    │
│  Динамик R: ⌀70мм, X=375мм, Y=400мм    │
├─────────────────────────────────────────┤
│  Принтер: 65×15мм, центр, Y=300мм      │
└─────────────────────────────────────────┘

Обработка:
1. Разметить все вырезы
2. Просверлить технологические отверстия в углах
3. Вырезать окна лобзиком
4. Скруглить углы R=3мм
5. Обработать кромки
```

### 3. Задняя Панель (Съёмная)

```
Размер: 1650 × 540 мм

Особенности:
- Съёмная (на петлях или магнитах)
- Вентиляционные отверстия внизу и вверху

Вентиляция:
- Нижняя решётка: 400×50мм, Y=50мм от низа
- Верхняя решётка: 400×50мм, Y=50мм от верха
- Можно заменить перфорацией ⌀10мм с шагом 20мм

Кабельные отверстия:
- Вход питания: ⌀30мм, Y=100мм, X=100мм
```

### 4. Основание

```
Размер: 500 × 540 мм

Функция: Нижняя платформа корпуса

Обработка:
- Прямоугольный рез без особенностей
- Возможны отверстия для кабелей снизу
```

### 5. Панель Управления

```
Размер: 504 × 300 мм
Угол установки: 12° от горизонтали

Вырезы (координаты от переднего левого угла):
┌─────────────────────────────────────────┐
│                                         │
│  Большая кнопка: ⌀62мм                 │
│  X=252мм (центр), Y=80мм               │
│                                         │
│  Кнопка L: ⌀30мм, X=112мм, Y=80мм      │
│  Кнопка R: ⌀30мм, X=392мм, Y=80мм      │
│                                         │
│  Клавиатура: 55×75мм углубление 5мм    │
│  X=224мм, Y=180мм                       │
│                                         │
│  LCD: 85×40мм окно                      │
│  X=122мм, Y=195мм                       │
└─────────────────────────────────────────┘

Обработка:
1. Разметить все позиции
2. Сверлить отверстия коронками (30мм, 62мм)
3. Фрезеровать углубление под клавиатуру (глубина 5мм)
4. Вырезать окно LCD
5. Скос торца 12° (опционально, можно при сборке)
```

### 6. Секция Маркиза

```
Размер: 540 × 350 мм
Установка: Под углом ~20° (верх корпуса)

Вырезы:
- Отверстие для камеры: ⌀30мм или 30×25мм
- Позиция: центр по ширине, 50мм от переднего края

Обработка:
- Скос переднего торца под угол крыши (опционально)
```

### 7-8. Внутренние Полки

```
Полка основания: 464 × 440 мм
- Для установки блока питания и Raspberry Pi
- Крепление: уголки или пазы в боковых стенках

Средняя полка: 464 × 200 мм
- Для установки NovaStar T50
- Высота установки: ~500мм от основания
```

## Инструмент для Раскроя

### Основной
- Циркулярная пила с направляющей шиной
- Электролобзик (криволинейные резы)
- Фрезер с копировальной втулкой

### Для Отверстий
- Коронка ⌀30мм (аркадные кнопки)
- Коронка ⌀62мм (большая кнопка)
- Коронка ⌀70мм (динамики)
- Сверло Форстнера для углублений

### Отделка
- Шлифмашина / наждачная бумага P120-P240
- Кромочный фрезер (снятие фасок)

---

## Порядок Раскроя

### Этап 1: Продольные Резы
1. Разделить лист 1 на полосы для боковых панелей
2. Разделить лист 2 на зоны для остальных деталей

### Этап 2: Поперечные Резы
1. Отделить прямоугольные заготовки
2. Оставить припуск 2-3мм на чистовую обработку

### Этап 3: Профильные Резы
1. Боковые панели — по шаблону лобзиком
2. Проверить симметричность двух панелей

### Этап 4: Вырезы и Отверстия
1. Разметить все вырезы на деталях
2. Выполнить отверстия коронками
3. Вырезать окна лобзиком
4. Фрезеровать углубления

### Этап 5: Финишная Обработка
1. Шлифовка всех кромок
2. Снятие фасок 2-3мм
3. Проверка сопряжения деталей

---

## Контроль Качества

- [ ] Все размеры соответствуют чертежам (±1мм)
- [ ] Углы резов точные (проверить угольником)
- [ ] Кромки без сколов и задиров
- [ ] Отверстия соосны и без эллипса
- [ ] Детали сопрягаются без зазоров
- [ ] Боковые панели симметричны

---

## Примечания

1. **Перед раскроем** проверьте фактические размеры всех компонентов (кнопки, экран, динамики)
2. **Припуск на пилу** — 3-4мм между деталями при разметке
3. **Направление волокон** — желательно вдоль длинной стороны детали
4. **Маркировка** — подписывайте детали карандашом сразу после резки
