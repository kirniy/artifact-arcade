# ARTIFACT — Чертёж Корпуса Аркадного Автомата

## Обзор Проекта

Автономный напольный аркадный автомат для гадания с AI-интеграцией. Корпус изготавливается из фанеры 18мм и содержит LED-дисплей, бегущую строку, панель управления, термопринтер и камеру.

---

## Габаритные Размеры

| Параметр | Значение | Примечание |
|----------|----------|------------|
| **Ширина** | **540 мм** | Определяется шириной бегущей строки (480мм) + зазоры + стенки |
| **Глубина** | **500 мм** (основание) → **350 мм** (верх) | Сужается кверху |
| **Высота** | **1750 мм** | Оптимально для роста 150-190 см |
| **Высота панели управления** | **950 мм** от пола | Эргономичная высота |
| **Центр экрана** | **~1200 мм** от пола | Комфортный угол обзора |

## Эргономические Стандарты

### Применённые Нормы

| Параметр | Значение | Источник |
|----------|----------|----------|
| Высота панели управления | 950 мм | 91-107 см — оптимум для взрослых стоя |
| Наклон панели управления | **12°** | 10-15° — комфортный угол для запястья |
| Наклон экрана | **20°** назад от вертикали | 60-75° от горизонтали для обзора |
| Диапазон роста пользователей | 150-190 см | Универсальный доступ |

### Обоснование Углов

**Почему 12° для панели управления?**
- Исследования показывают оптимум 10-15°
- Снижает нагрузку на запястье при длительном использовании
- Обеспечивает хороший доступ к кнопкам и клавиатуре

**Почему 20° для экрана?**
- Оптимальный угол обзора для различного роста
- Снижает блики от верхнего освещения
- Центр экрана на уровне глаз среднего пользователя

## Внутренняя Компоновка Электроники

### Нижний Отсек (0-400 мм от пола)

**Блок питания Mean Well LRS-200-5:**
- Позиция: Задняя центральная часть основания
- Зазор: 50 мм сверху для вентиляции
- Крепление: На платформу основания

**Raspberry Pi 4:**
- Позиция: Боковая стенка или полка
- Доступ: Через заднюю панель
- Зазор: 30 мм для кабелей и охлаждения

### Средняя Секция (400-900 мм — за панелью управления)

**NovaStar T50 (отправитель):**
- Позиция: На внутренней полке
- Рядом с точкой входа HDMI-кабеля
- Ethernet-кабель к DH418

**USB-хаб (при необходимости):**
- Рядом с Pi для подключения кнопок/клавиатуры

### Секция Экрана (900-1400 мм)

**NovaStar DH418 (приёмник):**
- Позиция: За сборкой LED-панелей
- По центру за сеткой 2×2 панелей
- Короткие HUB75E кабели к каждой панели

**Монтаж LED-панелей:**
- Алюминиевые уголки или деревянная рамка
- 4× P3 панели в сетке 2×2
- DH418 крепится к монтажной раме

### Внутренние Полки

1. **Полка основания:** 464×440 мм (БП + Pi)
2. **Средняя полка:** 464×200 мм (NovaStar T50)

## Маршрутизация Кабелей

```
Силовые кабели:     Вдоль основания, вверх по задней стенке
HDMI:               Pi → T50 (короткий участок)
Ethernet:           T50 → DH418 (через среднюю секцию)
GPIO шлейф:         Pi → Панель управления (клавиатура, кнопки)
USB:                Pi → Большая кнопка, динамики
CSI шлейф:          Pi → Камера (вверх через корпус)
Питание LED:        БП → Панели (отдельные линии 5В)
```
